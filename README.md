## ğŸ“– Usage Guide

### Creating a Room
1. Navigate to the homepage
2. Click **"Create Room"**
3. Share the generated Room ID with collaborators

### Joining a Room
1. Enter the Room ID provided by the host
2. Click **"Join Room"**
3. Start collaborating!

### Using the Code Editor
- Type code in real-time with your team
- Select a programming language from the dropdown
- Click **"Run Code"** to execute
- View output in the console panel

### Video Communication
- Click the video icon to enable your camera
- Click the microphone icon to toggle audio
- Use screen share to present your work

### Whiteboard
- Click the whiteboard tab
- Use drawing tools to explain concepts
- Save or export your diagrams
